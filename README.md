# ShiftPay

Shift salary calculator — installable PWA (iPhone & Android) + Android APK.

- **Install page:** `install.html` (auto-detects iPhone vs Android)
- **Web app / iPhone:** open `index.html` in Safari → Share → Add to Home Screen
- **Android app:** download `ShiftPay.apk`

Works offline · multilingual (EN/HE/AR/RU) · data stays on the device.
